# 🏨 Hotel Management System (Java + Swing + MySQL)

## 📌 Project Overview

Hotel Management System ek desktop-based application hai jo hotels ke daily operations ko manage karne ke liye banaya gaya hai. Is project me user rooms add kar sakta hai, booking kar sakta hai, aur checkout process handle kar sakta hai — sab kuch ek simple GUI ke through.

---

## 🚀 Features

* 🔐 Login System
* 🏠 Room Management (Add Rooms)
* 📅 Room Booking System
* 📋 View Available Rooms
* 📊 View All Bookings
* 💳 Checkout & Payment System
* 🛠 Admin Panel

---

## 🛠 Technologies Used

* **Java (Swing)** – GUI Design
* **MySQL** – Database
* **JDBC** – Database Connectivity
* **NetBeans IDE** – Development Tool

---

## 🧩 Project Structure

* `DBConnection.java` → Database connection
* `Login.java` → User login
* `Dashboard.java` → Main navigation panel
* `BookRoom.java` → Room booking system
* `ViewRooms.java` → Show all rooms
* `ViewBook.java` → Show bookings
* `Checkout.java` → Payment system
* `AdminPanel.java` → Admin controls
* `AddRoom.java` → Add new rooms

## 💡 Key Functionalities

### 🏨 Room Booking

User room type select karta hai (Single/Double) aur available rooms dropdown me show hote hain.

### 💳 Checkout

Sirf “Not Paid” bookings show hoti hain aur payment ke baad status “Paid” ho jata hai.

### 🛠 Admin Panel

Admin naye rooms add kar sakta hai predefined price options ke saath (₹1000 – ₹10000).
